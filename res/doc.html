Implementace

V adresári app sa nachádzi implementace rest api a také trieda databáze, která poskytuje rozhraní medzi rest api a databází. Ako php rozhraní pre prácu s mysql databázou sme se rozhodli zvoliť PDO.

Back-end

Rozhranie databaze

Rozhranie pre pracu s databazou je definovane v subore . Jedna sa o PHP triedu, ktora zapuzdruje pracu s databazou tak, aby priame dotazy mimo tohoto suboru neboli nutne. Avsak pre pripad, ze by to bolo potrebne, existuje funkcia , ktora vrati referenciu na objekt.
Cela funkcnost naseho API je obsiahnuta v tejto triede.
Kazda funkcia ma tu istu strukturu:

  • Definovanie navratovej premennej
  • blok, ktory zachyti pripadne errory s jednotlivymi dotazmi
  • Samotne SQL dotazy
  • Databazove transakcie ak vkladame data do databazy
  • Vratenie premennej
